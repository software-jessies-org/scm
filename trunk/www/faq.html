SCM FAQ

I've installed SCM. Now what?

If you use Evergreen, SCM is automatically integrated into the SCM menu.

Cygwin users should already have "checkintool" and "revisiontool" on their path, because our installer modifies their path.

Linux users should already have "checkintool" and "revisiontool" on their path, because we install links in /usr/bin.

Mac OS users will need to ensure that the Contents/Resources/scm/bin/ directory in SCM.app is on their path. You can use salma-hayek's "LSFindApplicationForInfo org.jessies.scm" to find the application at run time, but we have yet to write a relocatable script/binary to do this for you.

Solaris users should already have "checkintool" and "revisiontool" on their path, because we install links in /usr/bin.
