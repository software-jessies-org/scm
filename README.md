This directory contains the scm package, currently comprising
these front-ends:

RevisionTool | revision history browser
CheckInTool  | check-in tool

And these back-ends:

* Bazaar
* BitKeeper
* CVS
* Git
* Mercurial
* Subversion

word "SCM" in your Subject: line.

Enjoy!
